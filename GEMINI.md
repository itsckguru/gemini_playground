## âš¡ Prompt Engineering Cheat Sheet

When crafting queries in this playground, aim for the **C.R.E.F.** method to get the best results:

* **C - Context:** Who is Gemini? What is the background?
    * *Example:* "Act as a Senior Python Engineer..."
* **R - Request:** What specifically do you need? Be granular.
    * *Example:* "Refactor this function to be more memory efficient..."
* **E - Examples:** Give Gemini a shot of what you want (Few-Shot Prompting).
    * *Example:* "Input: A, Output: B. Now do it for C."
* **F - Format:** How do you want the output?
    * *Example:* "Output as a markdown table," or "JSON format only."

> **Tip:** If the output isn't right, treat it like a conversation. iterate with "Refine that, but remove X and add Y."
